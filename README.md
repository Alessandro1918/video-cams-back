## 🗂️ Utilização

### 🐑🐑 Clonando o repositório:

```bash
  $ git clone url-do-projeto.git
```

### ▶️ Rodando o App:

```bash
  $ cd video-cams-back
  $ npm install             #download dependencies to node_modules
  $ npm run dev             #start the project
  Server running on http://localhost:4000
```
